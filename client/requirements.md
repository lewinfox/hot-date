## Packages
date-fns | Essential for calendar generation, date math, and formatting
framer-motion | For premium minimal animations (page transitions, staggered list reveals)
lucide-react | Base stack icon library, but specifically noting it for UI elements
clsx | Utility for conditional classes
tailwind-merge | Utility for merging tailwind classes safely

## Notes
- Expecting standard REST API endpoints as defined in routes_manifest
- Event slugs are used as the primary identifier in URLs (/event/:slug)
- Calendar renders 3 months starting from the current month
